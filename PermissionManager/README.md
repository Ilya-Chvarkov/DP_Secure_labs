# Permission Manager

Инструмент для сохранения и восстановления файлов/каталогов вместе с их правами доступа (включая ACL).

## Возможности

-  Сохранение файлов и целых каталогов в один ZIP-архив
-  Восстановление содержимого и прав доступа
-  Поддержка ACL (на Linux через getfacl/setfacl)
-  Кроссплатформенность (Windows, Linux, macOS)
-  Графический интерфейс (GUI) и командная строка (CLI)

## Быстрый старт

### Запуск GUI

python permission_manager.py

# или
